# MikeTyping-onnx 🎙️

Голосовой ввод текста в любом приложении Windows.
Нажмите **Scroll Lock** → говорите → нажмите **Scroll Lock** → текст вставлен.

## Особенности
- Работает офлайн, без интернета
- Русский и английский (авто-переключение по раскладке клавиатуры)
- Автоматическая ставка через Ctrl+V — работает в любом поле ввода
- Звуковой сигнал после загрузки моделей — знаете, когда можно начинать

## Установка

1. Скачайте архив с [последнего релиза](ссылка).
2. Распакуйте в любую папку (желательно на SSD).
3. Запустите `install.bat` (один раз) — он создаст виртуальное окружение и установит библиотеки.
4. После установки на рабочем столе появится ярлык **MikeTyping**. Запустите его.
5. Чтобы закрыть приложение - ещё раз запустите ярлык на рабочем столе.

**Важно:** при первом запуске через ярлык прозвучит двойной звуковой сигнал — это значит, модели загружены и можно пользоваться.

## Использование

- Откройте любое приложение (Word, браузер, Telegram).
- Нажмите `Scroll Lock` — услышите короткий сигнал, на экране (если запускали через `start.bat` без `pythonw`) появится статус `АКТИВНА 🎙️`.
- Произнесите фразу.
- Снова нажмите `Scroll Lock` — текст распознается и автоматически вставится в текущее поле.

## Настройка

Все параметры в `src/config.py`:
- `HOTKEY` — можно сменить на другую клавишу (например `Key.f13`)
- `AUTO_SPACE` — добавлять пробел после вставки (`True`/`False`)
- `ENABLE_VAD` — включить подавление шума (требует `models/vad.onnx`)

## Требования

- Windows 10 / 11
- Python 3.12+ (при установке обязательно отметить "Add Python to PATH")
- 1 Гб место на диске.
- 250 Мб оперативной памяти.

## Устранение проблем

- **Не вставляется текст** → проверьте, что активное окно принимает Ctrl+V (например, консоль или терминал не поддерживают)
- **Нет звука после запуска ярлыка** → проверьте, что в системе включены звуки, активируйте окружение `.venv` и запустите `python main.py` вручную, (появится консоль с логами)
- **Ошибка при установке** → запустите `install.bat` от имени администратора
