# MikeTyping-onnx 🎙️

Голосовой ввод текста в любом приложении Windows.
Нажмите **Scroll Lock** → говорите → нажмите **Scroll Lock** → текст вставлен.

## Особенности
- Работает **офлайн**, без интернета.
- **Русский и английский** (авто-переключение по раскладке клавиатуры).
- **Автоматическая вставка** через Ctrl+V — работает в любом поле ввода.
- **Звуковой сигнал** после загрузки моделей — знаете, когда можно начинать.
- **Контекстное усиление (hotwords)** для повышения точности распознавания имён, терминов и редко встречающихся слов (только для русского языка).
- **VAD (Voice Activity Detection)** — фильтрация тишины и фонового шума (опционально, требует модели `vad.onnx`).
- **Умная расстановка запятых** на основе синтаксического анализа (Dependency Parsing) — запятые автоматически добавляются перед относительными (`который`, `что`, `чей` и др.) и противительными (`а`, `но`, `однако`) союзами, а также между однородными членами предложения.
- **Автоматическая постобработка текста** — исправление регистра (первая буква заглавная), добавление точки в конце фразы, удаление лишних пробелов и идеальная сборка токенов без багов со знаками препинания.
- **Подробное логирование** — при включённом режиме `DEBUG = True` все события пишутся в файл `logs/debug.log` (отдельно от кода программы).
- **Удобный запуск и остановка** — ярлык на рабочем столе запускает приложение, повторный запуск того же ярлыка — останавливает, с системными уведомлениями о статусе.

## Установка

1. **Скачайте** архив с [последнего релиза](https://github.com/AITISPEC/MikeTyping-onnx/releases).
2. **Распакуйте** в любую папку (желательно на **SSD**).
3. **Запустите** `install.bat`.
   Скрипт проверит версию **Python (требуется 3.12+)** и при отсутствии укажет ссылки на скачивание.
4. После установки **на рабочем столе появится ярлык MikeTyping**. Запустите его.
5. **Чтобы закрыть приложение** — ещё раз запустите ярлык на рабочем столе (появится уведомление «MikeTyping остановлен»).

**Важно:** при запуске прозвучит двойной звуковой сигнал — это значит, модели загружены и можно пользоваться.

## Использование

- **Откройте любое приложение**. Установите курсор в поле для ввода текста.
- **Нажмите `Scroll Lock`** — услышите короткий сигнал.
- **Произнесите фразу**.
- **Снова нажмите `Scroll Lock`** — текст распознается, пройдёт автоматическую расстановку запятых, капитализацию и получит точку в конце, после чего вставится в текущее поле. Прозвучит однократный сигнал об окончании распознавания.

## Настройка

Все параметры в `src/config.py`:

| Параметр | Описание |
|----------|----------|
| `HOTKEY` | Горячая клавиша (по умолчанию `Key.scroll_lock`). Можно изменить на любую из `pynput.keyboard.Key`. |
| `AUTO_SPACE` | Добавлять пробел после вставки (`True`/`False`). |
| `AUTO_SWITCH_BY_KEYBOARD_LAYOUT` | Автоматически переключать язык ASR в зависимости от раскладки клавиатуры (только RU/EN). |
| `ENABLE_VAD` | Включить подавление тишины (требует `models/vad.onnx`). |
| `VAD_THRESHOLD` | Порог чувствительности VAD (0.0–1.0). Для тихих микрофонов рекомендуется 0.05 и ниже. |
| `USE_HOTWORDS` | Включить контекстное усиление (русский язык). |
| `HOTWORDS_SCORE` | Вес ключевых слов (1.0–3.0). |
| `HOTWORDS_RU_FILE` | Путь к файлу `hotwords.txt` (одно слово на строку). |
| `BLOCK_SIZE` | Размер аудиоблока. При проблемах с VAD можно увеличить (например, до 7056). |
| `USE_TEXT_POSTPROCESSING` | Включить постобработку текста: расстановку запятых, заглавную букву и точку в конце. |
| `DEBUG` | Включить подробное логирование в `logs/debug.log`. |

### Hotwords (контекстное усиление)

1. Убедитесь, что в папке с русской моделью (`models/asr/ru/`) есть файлы `bpe.vocab` и `hotwords.txt`.
2. В `hotwords.txt` запишите важные слова (по одному на строку), например:
   ```text
   НЕЧОСИ
   НЕЙРОФАКУЛЬТЕТ
   МАЙКТАЙПИНГ
   ```
3. В `config.py` установите `USE_HOTWORDS = True` и при необходимости измените `HOTWORDS_SCORE` (чем выше, тем сильнее модель будет тяготеть к этим словам).

- 500 Мб SSD
- 1 Гб ОЗУ

## Устранение проблем

- Обязательно в src/config.py установите флаг **DEBUG = True**, чтобы увидеть подробную информацию в логах.
- **Не вставляется текст** → проверьте, что активное окно принимает Ctrl+V (например, консоль или терминал не поддерживают).
- **Нет звука после запуска ярлыка** → проверьте, что в системе включены звуки, активируйте окружение `.venv` и запустите `python main.py` вручную (появится консоль с логами).
- **Ошибка при установке** → запустите `install.bat`, убедитесь, что Python 3.12+ установлен и добавлен в PATH.
- **Авто-переключение языка не работает** → убедитесь, что `AUTO_SWITCH_BY_KEYBOARD_LAYOUT = True`, а раскладка клавиатуры — русская (0x419) или английская (0x409). При использовании других раскладок программа выдаст предупреждение в консоль и лог.
- **Нет запятых в распознанном тексте** → проверьте, что в `config.py` установлен `USE_TEXT_POSTPROCESSING = True` и установлены все зависимости (`pip install natasha pymorphy3`). Модуль пунктуации требует интернет-соединения при первом запуске для загрузки языковых моделей Natasha.
